Introduction

This driver is intended to provide an API functions to ease the development of applications.

Overview

This document is intended to help in the development of applications based on VL53LX sensor and describes the API of the driver delivered by ST as open source C code.
Some of the API files are hardware and platform dependent (specially I2C access) so need for end user adaptation according to his own platform.

Device Info from API

The API provide a function that can be used to obtain information of the device used like the cut version. This function is VL53LX_GetDeviceInfo().

Coding Standards

The implementation of this API will follow Linux Kernel rules as defined in https://www.kernel.org/doc/Documentation/CodingStyle
