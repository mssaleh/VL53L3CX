RangeStatus

The Range Status is contained in the VL53LX_RangingMeasurementData_t and give the quality of the latest ranging.

This is a 8 bit data which contains the following fields:

VL53LX_RANGESTATUS_RANGE_VALID = Range Valid

This value indicate that the ranging is valid.

VL53LX_RANGESTATUS_SIGMA_FAIL = Sigma Fail

This value indicate that the sigma limit check has failed. Use the function VL53LX_SetLimitCheckEnable() and VL53LX_SetLimitCheckValue() to manage the limit.

VL53LX_RANGESTATUS_SIGNAL_FAIL = Signal Fail

This value indicate that the signal check has failed. This can happens when there is no target or when the Range Ignore threshold check has failed. Use the function VL53LX_SetLimitCheckEnable() and VL53LX_SetLimitCheckValue() to manage the limit.

VL53LX_RANGESTATUS_RANGE_VALID_MIN_RANGE_CLIPPED = Min Range Fail

This value indicate that the min range check has failed.

VL53LX_RANGESTATUS_OUTOFBOUNDS_FAIL = Phase out of valid limits

This value indicate that the Phase check has failed.

VL53LX_RANGESTATUS_HARDWARE_FAIL = HardWare Fail

This value indicate that the Hardware check has failed.

VL53LX_RANGESTATUS_RANGE_VALID_NO_WRAP_CHECK = Range Valid no Wrap around check

This value indicate that range measurements exist but the check for the wrap around has not been done.

VL53LX_RANGESTATUS_WRAP_TARGET_FAIL = wrapped target

Wrapped target - no matching phase in other VCSEL period timing.

VL53LX_RANGESTATUS_SYNCRONISATION_INT = ACK after VL53LX_StartMeasurement() call

Very first interrupt received when starting ranging in back to back mode. Application shall ignore the measurement data.

VL53LX_RANGESTATUS_RANGE_VALID_MERGED_PULSE = Range Quality Level information

Warns the application that ranging is ok but object reported is result of multiple pulses merging together.

VL53LX_RANGESTATUS_TARGET_PRESENT_LACK_OF_SIGNAL = Range Quality Level information

this warning is different to phase fail.

VL53LX_RANGESTATUS_NONE = None

No Update. Tipically this happens when the data get doesn't contains meaningful values
